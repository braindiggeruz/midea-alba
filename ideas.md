## Вариант 1: "Dark Cyber Tech" — Киберпанк-технологии

**Design Movement:** Cyberpunk Tech / Dark UI

**Core Principles:**
- Тёмный фон (#050D1A) с неоновыми акцентами cyan (#00E5FF) и жёлтым (#FFD600)
- Гексагональные паттерны как декоративные элементы (как в постах)
- Агрессивная типографика — огромные заголовки, контрастные CTA
- Данные и цифры как главные продающие элементы

**Color Philosophy:**
- Background: #050D1A (почти чёрный синий)
- Primary Accent: #00E5FF (неоновый cyan — технологичность)
- CTA Color: #FFD600 (жёлтый — срочность, деньги)
- Secondary: #1A2744 (тёмно-синий для карточек)
- Text: #FFFFFF / #B0C4DE

**Layout Paradigm:**
- Асимметричный hero с кондиционером справа и текстом слева
- Секции с наклонными разделителями (clip-path)
- Плавающие гексагоны как фоновые декорации
- Sticky CTA-кнопка в хедере

**Signature Elements:**
- Неоновые гексагоны с иконками фич (как в постах)
- Анимированные circuit-board линии на фоне
- Countdown timer для срочности

**Interaction Philosophy:**
- Hover на гексагонах — неоновое свечение
- Параллакс-эффект на hero-секции
- Smooth scroll с анимацией появления секций

**Animation:**
- Entrance: fade-in + slide-up для текста
- Гексагоны: pulse glow animation
- Цена: countup animation
- CTA кнопка: shimmer effect

**Typography System:**
- Display: Bebas Neue (огромные заголовки)
- Body: Montserrat (читаемость)
- Accent: Orbitron (технические данные)

## Вариант 2: "Electric Storm" — Электрический шторм

**Design Movement:** High-Energy Sales / Electric Brutalism

**Core Principles:**
- Максимальная агрессия продаж — каждый элемент кричит "КУПИ"
- Электрические молнии и энергетические эффекты
- Жёлто-зелёная палитра (деньги + энергия)
- Brutalist typography — сломанные сетки, наклонные элементы

**Color Philosophy:**
- Background: #0A0A0A (чистый чёрный)
- Primary: #39FF14 (неоновый зелёный — энергия)
- CTA: #FFE500 (жёлтый — деньги)
- Accent: #FF6B00 (оранжевый — срочность)
- Text: #FFFFFF

**Layout Paradigm:**
- Зигзагообразный layout — секции чередуются слева-справа
- Диагональные разрезы между секциями
- Огромные числа как декоративные элементы

**Signature Elements:**
- SVG молнии между секциями
- Анимированный спидометр (как в посте "охлаждает за 3 мин")
- Стикеры-бейджи с ценами

**Interaction Philosophy:**
- Shake animation на CTA при hover
- Электрические разряды при скролле
- Flicker effect на заголовках

**Animation:**
- Lightning bolt animations
- Text glitch effect
- Explosive entrance animations

**Typography System:**
- Display: Black Han Sans (корейский bold)
- Body: Noto Sans (мультиязычность)
- Numbers: Oswald Extra Bold

## Вариант 3: "Deep Space Premium" — Космический премиум

**Design Movement:** Space Tech / Premium Dark

**Core Principles:**
- Глубокий космический фон с звёздами и туманностями
- Cyan/blue неоновые акценты — технологичность и холод
- Премиальные glassmorphism карточки
- Данные и факты как главные аргументы

**Color Philosophy:**
- Background: #020818 (глубокий космос)
- Primary: #06B6D4 (cyan — холод, технологии)
- CTA: #FBBF24 (золотой — ценность)
- Cards: rgba(255,255,255,0.05) (glassmorphism)
- Text: #F0F9FF / #94A3B8

**Layout Paradigm:**
- Центрированный hero с кондиционером в центре
- Радиальные линии от центра (как в постах)
- Floating cards с характеристиками

**Signature Elements:**
- Particle system на фоне (звёзды)
- Радиальный gradient glow вокруг кондиционера
- Animated temperature counter (45°C → 22°C)

**Interaction Philosophy:**
- Smooth parallax на hero
- Glow effect на картах при hover
- Scroll-triggered counter animations

**Animation:**
- Particle floating animation
- Counter animations для цифр
- Fade-in с blur для секций

**Typography System:**
- Display: Russo One (технологичность)
- Body: Nunito (мягкость, читаемость)
- Data: Space Mono (технические данные)
